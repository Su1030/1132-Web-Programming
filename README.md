## 📖 會員管理系統 - Flask 專案
這是一個基於 Python Flask 開發的輕量級會員管理系統，支援完整的人員登入、註冊、資訊編輯以及管理員後台功能。專案採用語意化設計，兼顧開發效率與使用者體驗。

---

## ✨ 功能特點

### 🔐 帳戶與安全管理
*   **強化的註冊系統 (`/register`)**：
    *   **嚴格驗證**：Email 格式檢查（需含 `@` 與 `.`）、密碼長度限制（6-16 字元）。
    *   **彈性欄位**：電話採**選填**制，若輸入則需符合 `09` 開頭之 10 碼格式。
    *   **防重機制**：自動比對 `users.json` 確保帳號與 Email 唯一性。
*   **管理者權限邏輯**：
    *   系統識別 `admin` 帳號為最高權限者，動態顯示「會員管理」後台按鈕。
    *   內建保護機制：禁止刪除管理者帳號或目前登入之帳號。

### 📊 資料表現力
*   **管理員後台**：以表格形式管理會員，支援即時編輯電話、生日或重設密碼。
*   **自定義過濾器**：
    *   **隱私遮罩**：電話號碼自動轉換為 `0912****78` 格式。
    *   **日期本地化**：西元日期自動計算並轉換為「民國年」顯示。

### 🎨 互動體驗
*   **響應式佈局**：基於 Pico.css V2，完美適配手機與桌面端。
*   **優化流程**：註冊成功導向專屬 `welcome.html`；邏輯錯誤引導至統一錯誤頁面。
*   **置中按鈕佈局**：所有表單控制按鈕均採用 Flexbox 置中排列，提升視覺穩定感。

---

## 🛠️ 技術棧

*   **後端**: Python 3.x / Flask
*   **前端**: Jinja2 / Pico.css (V2) / 現代化 Flexbox 佈局
*   **資料儲存**: JSON 檔案系統 (`users.json`)

## 🚀 安裝與執行步驟

### 1\. 複製儲存庫

```bash
git clone https://github.com/你的帳號/Flask-UserHub.git
cd Flask-UserHub
```

### 2\. 安裝必要套件

```bash
pip install -r requirements.txt
```

### 3\. 執行程式

```bash
flask --debug run
```
